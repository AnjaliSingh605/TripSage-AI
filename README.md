# TripSage AI – Smart Travel Planner

TripSage AI is a web application that helps users plan trips using AI. The platform collects travel details such as destination, budget, and trip duration, and generates a structured travel itinerary with recommendations.

## Features

* AI-based travel itinerary generation
* Interactive flow to collect trip preferences
* Personalized travel recommendations
* Responsive and modern user interface

## Tech Stack

* Next.js
* React
* Tailwind CSS
* TypeScript
* AI APIs
* Vercel (Deployment)

## Run Locally

Clone the repository

```bash
git clone https://github.com/AnjaliSingh605/TripSage-AI.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open **http://localhost:3000** in your browser.
